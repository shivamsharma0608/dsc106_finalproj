# Data

Download once (≈9 MB):

```bash
curl -L -o regular_season_totals_2010_2024.csv \
  https://raw.githubusercontent.com/NocturneBear/NBA-Data-2010-2024/main/regular_season_totals_2010_2024.csv
```

**Also for Visual 1 (1996–2023):**

```bash
curl -L -o team_stats_traditional_rs.csv \
  https://raw.githubusercontent.com/Brescou/NBA-dataset-stats-player-team/main/team/team_stats_traditional_rs.csv
```
